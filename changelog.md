# Changelog

## [1.0.0] - 2025-12-08

### Adicionado
- Suporte a renderização de Markdown nas respostas do bot
- Comportamento de quebra de linha com Enter e envio com Shift+Enter (similar ao WhatsApp)
- Limpeza automática de espaços extras e linhas vazias nas mensagens
- Nome do bot configurável via variável de ambiente no backend
- Versão da aplicação definida como 1.0.0

### Corrigido
- Espaçamentos desnecessários nos balões de fala
- Problemas de overflow e formatação de mensagens
- Balões de fala sumindo devido a conflitos de CSS

### Melhorado
- Visual dos balões de fala para parecer mais com WhatsApp
- Espaçamento interno reduzido para layout mais compacto
- Remoção de margens extras em elementos Markdown

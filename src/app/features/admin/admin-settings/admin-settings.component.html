📚 Como conectar o N8N e Hotmart

🔄 Automação no N8N (Configuração do Prompt)

Sua automação do N8N receberá o seguinte payload JSON a cada nova mensagem:

{{'{'}}
  "userId": 12,
  "username": "Maicon",
  "message": "Olá! Gostaria de saber os preços",
  "systemPrompt": "Você é o assistente virtual da..."
{{'}'}}

Para ler o prompt separado do chat, no nó do seu N8N (como um AI Agent ou OpenAi LLM), mapeie o campo System Message / Prompt para ler {{'{{'}}$json.systemPrompt{{'}}'}} e a mensagem do chat para {{'{{'}}$json.message{{'}}'}}.

🛒 Integração Hotmart (Criação de Clientes)

Acesse o painel da Hotmart → Ferramentas → Webhooks (API e Serviços):

  1. Adicione um novo webhook.
  2. Cole a URL de Integração exibida acima.
  3. Adicione um Cabeçalho (Header) Customizado com a chave x-webhook-token e o valor do Token de Segurança.
  4. Selecione os eventos de interesse (ex: Compra Aprovada).
